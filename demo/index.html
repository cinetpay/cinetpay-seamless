CinetPay Seamless

Demo — paiement inline sans redirection

Pour obtenir un paymentToken, initialisez un paiement via votre backend en appelant l'API CinetPay POST /v1/payment.

Vous pouvez utiliser cinetpay-js (Node.js), cinetpay-laravel-sdk (PHP), ou un appel API direct depuis n'importe quel langage.
